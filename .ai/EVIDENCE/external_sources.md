# External Sources Registry

This registry tracks all external repositories and forks cloned into the `external_repos/` workspace.

| REPO_NAME | SOURCE_URL | PURPOSE | STATUS | ADDED_DATE |
|---|---|---|---|---|
| (Example) `adk-docs` | `https://github.com/google/adk-docs` | Evidence/Reference | ACTIVE | 2026-02-28 |

*When adding a new repository to `external_repos/`, add a new row to this table.*
