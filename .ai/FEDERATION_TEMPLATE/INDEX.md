# FEDERATION INDEX (TEMPLATE)
Purpose: Hubless multi-repo coordination via links + short status exports.
Key files:
- peers.md (peer repo pointers)
- status_export.md (local status, <= 25 lines)
Usage:
When a new repository is added to `external_repos`, it should copy this folder to the root of its own `/.ai/` folder, and rename it to `FEDERATION`.
