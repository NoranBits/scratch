# External Repositories Workspace

This directory (`external_repos/`) is the designated location for cloning and storing external GitHub forks and repositories.

## Purpose

These external sources may be used for:
- Personal customization trials before merging upstream.
- Analyzing external projects as evidence for code quality standards.
- Incorporating external libraries or codebases as part of the broader project.

## Governance

All repositories added to this directory MUST be registered in the central governance plane. 
When you add a new repository here, you must update:
`/.ai/EVIDENCE/external_sources.md`

This ensures that the Anti-Gravity orchestrators and Field Agents are aware of these external dependencies and can track their purpose and status correctly.
